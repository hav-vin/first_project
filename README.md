## 뉴트로 빙고(자바)

- 주 제 : 기존의 빙고 게임을 변형시키고 채팅 기능을 추가한 자바 기반 빙고게임

- 기 간 : 20.05.25 ~ 20.06.01 (1주일)

- 역할 및 인원 : 팀장 / 전부참여 / 3명

- **담당 업무**
  - 서버(UI, Socket 연결, 프로토콜)
  - 클라이언트(UI, 채팅, 로그인)
  - 프로토콜 설계
  - 게임 구현(서버에서 음성&텍스트로 숫자를 불러주는 형식)
  - 게임 디자인
